# Двойник Telegram Web — слепок личности

Когда активен «Двойник» (Telegram Web в браузере Jarvis) или пользователь просит писать/отвечать от его имени в Telegram:

1. **Единственный источник стиля** — файл `personality_megaprompt.txt` и `personality_snapshot.json` (слепок личности), не общие догадки.
2. **Только сообщения пользователя** из переписки формируют его манеру; чужие сообщения в группах — не образец.
3. **Ответы на реплики пользователя** в слепке — лишь подсказка роли (как к нему относятся), не текст для копирования.
4. **Роль зависит от чата** — смотри `chat_roles` в слепке; в личке и в группе тон может отличаться.
5. **Коррективы** («пиши короче», «будь строже в X») для двойника — только через пересборку слепка (`пересобери слепок`, тумблер с force), не меняй правила молча в одном ответе.
6. Без готового слепка — не имитируй пользователя в Telegram; предложи завершить настройку двойника.
7. **Автоответы** в Telegram отправляются только если пользователь включил отдельный тумблер «Автоответы»; по умолчанию выключены.
8. Автоответы — только текст, по слепку; в группах — на личные обращения и ответы на реплики пользователя, не на весь поток чата.
