# Джарвис — голосовой ассистент для Windows

![License: MIT](https://img.shields.io/badge/лицензия-MIT-green)
![Free](https://img.shields.io/badge/стоимость-бесплатно-blue)
![Platform](https://img.shields.io/badge/платформа-Windows-lightgrey)

**Бесплатный открытый репозиторий** для развития проекта «Джарвис»: голосовые команды, управление ПК, интеграции и локальный веб-интерфейс.

> Этот репозиторий **не продаётся** и **не требует подписки**. Код, документация и сборки для личного использования публикуются **бесплатно**. Разработка ведётся сообществом и автором проекта.

---

## Что умеет Джарвис

| Возможность | Описание |
|-------------|----------|
| Голосовые команды | Распознавание речи и выполнение действий на компьютере |
| Управление ПК | Запуск программ, громкость, окна, системные действия |
| Веб-интерфейс | Настройка и мониторинг через браузер (`localhost`) |
| Расширяемость | Модули, сценарии, подключение внешних сервисов |

Подробнее об изданиях — в [README_EDITIONS.md](README_EDITIONS.md).

## Требования

| Параметр | Значение |
|----------|----------|
| ОС | **Windows 10/11** (64-bit) |
| Python | **3.10+** (для запуска из исходников) |
| Микрофон | Любой рабочий (для голосовых команд) |
| Интернет | Нужен для первой установки зависимостей и части функций |

## Быстрый старт

### Запуск из исходников

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Interfero/forgithub.git
   cd forgithub/jarvis
   ```
2. Установите зависимости и запустите — пошагово в [START.ru.txt](START.ru.txt).

### Готовая сборка (релиз)

1. Откройте вкладку **[Releases](https://github.com/Interfero/jarvis/releases)** (Релизы).
2. Скачайте архив **«Джарвис-Windows-x64.zip»** из последнего релиза.
3. Распакуйте и запустите `Jarvis.exe` (или скрипт из инструкции в релизе).

> **Совет:** в описании каждого релиза — список изменений, требования и известные проблемы на русском языке.

## Релизы и версии
