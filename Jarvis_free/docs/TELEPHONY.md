# Телефония — звонок на Джарвис через АТС

## Как это работает

1. Клиент звонит на ваш номер в облачной АТС.
2. АТС отправляет HTTP-запрос на Jarvis (`/api/telephony/webhook` или `/api/telephony/scenario`).
3. Jarvis синтезирует приветствие (голос edge-tts, «Джарвис») и отдаёт URL аудио.
4. АТС проигрывает файл абоненту. При DTMF/следующих событиях — краткий ответ DeepSeek (если включено).

## Обязательно

- **Публичный URL** (ngrok, Cloudflare Tunnel): АТС из интернета не достучится до `127.0.0.1`.
- **Подробно для Mango Office:** см. [MANGO_PUBLIC_URL.md](./MANGO_PUBLIC_URL.md)
- В панели разработчика → **АТС** укажите `https://ваш-туннель` без слэша в конце.
- Включите «АТС — включено», нажмите **Синтез речи**, **Сохранить**.

## Mango Office

1. Личный кабинет → Интеграция → API: **ключ** и **salt**.
2. В Jarvis: провайдер **Mango Office**, extension сотрудника, линия АОН.
3. Схема входящего звонка: шаг **«Запрос к внешней системе»** → URL:
   `https://ВАШ-ТУННЕЛЬ/api/telephony/scenario`
4. События звонков (опционально): `https://ВАШ-ТУННЕЛЬ/api/telephony/webhook`
5. Тест: **Позвонить** — исходящий звонок через Mango API на ваш мобильный.

## Zadarma

1. Настройки → API: key и secret.
2. «Уведомления о звонках» → URL: `https://ВАШ-ТУННЕЛЬ/api/telephony/webhook`
3. IVR с webhook: ответ `ivr_play` — укажите **ID файла** из кабинета или загрузите mp3 вручную.
4. Для динамической речи Jarvis загрузите синтезированный `greeting.mp3` в Zadarma и укажите ID в настройках.

## Generic

POST JSON на `/api/telephony/webhook`:

```json
{"event": "incoming", "call_id": "1", "from": "+79...", "to": "+74..."}
```

Ответ: `audio_url` для проигрывания в вашей АТС.
